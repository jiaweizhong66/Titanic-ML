# Titanic-ML

This a ML model that predicts passenger's survival rate
Trained and tested with the data from [here](https://www.kaggle.com/c/titanic/data)

## Tech Stack
```
Language: Python
Frameworks: Pandas, Numpy, scikit-Learn
```
